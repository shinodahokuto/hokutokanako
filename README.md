# hokutokanako

このリポジトリには結婚式の招待状テンプレートが含まれます。

使い方:
- 招待状のHTML: `invitation.html`
- スタイル: `styles.css`

背景画像を使いたい場合:
1. `assets/` フォルダを作成し、背景画像を `assets/background.jpg` に置いてください。
2. ブラウザで `invitation.html` を開くと背景画像が反映されます。

備考:
- メールリンクは `invitation.html` 内の `mailto:example@example.com` を編集してご利用ください。
- 印刷ボタンで印刷向けのスタイルが適用されます。
